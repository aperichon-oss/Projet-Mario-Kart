Mario Kart 8 Deluxe World Records

Update 2024-06-17
Due to the acceleration bug introduced in Wave 5, there are 3 200cc WRs that were deemed to be likely unbeatable on the current version of the game:
Toad's Turnpike, Paris Promenade, and London Loop
These WRs will stop receiving duration on 2024-06-17 and the accel bug WRs will start receiving duration and will count as the current WR on the affected courses.
This change is to allow competition to continue on these courses as there is no convenient way to time trial on older versions of the game.
See the MK8DX Patch Info page for more info.
